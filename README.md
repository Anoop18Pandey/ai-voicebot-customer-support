# AI Voice Bot – Customer Support Automation

## Overview
This project is an AI-powered voice bot that can handle basic customer support queries using speech interaction.

## Features
- Speech-to-Text using Whisper
- Intent Detection
- Response Generation
- Text-to-Speech using gTTS
- FastAPI API

## API Endpoint
POST /voicebot

## How to Run
pip install -r requirements.txt  
uvicorn main:app --reload
